Como Instalar (Getting Started)

Apenas dois arquivos separam você de um editor poderoso.

1 Inclua o jQuery e o Weaver

Adicione as tags <script> no <head> ou antes do fechamento do seu </body>.


<!-- jQuery (Requisito) -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Weaver Editor -->
<script src="dist/weaver-editor.min.js"></script>

2 Adicione um Textarea

Crie um campo de texto em sua página com um ID que você usará para inicializar.


<form method="post">
  <textarea id="meu-editor" name="conteudo"></textarea>
</form>

3 Inicialize o Editor

Chame a função weaver() passando as configurações desejadas (ou deixe vazio para o padrão).


$(document).ready(function() {
  $('#meu-editor').weaver({
    placeholder: 'Digite sua mágica aqui...',
    height: 400
  });
});
